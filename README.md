# dataR
Data project using R - Gathering, Filter and display data about electric vehicles and charge stations

To run the project
- Open the Rproject folder
it contains 4 R script : 
- global.R that load all the needed libraries and create the dataframes
- server.R
- ui.R
- app.R that import all the other scripts and run the entire app

the "www" folder contains only the image displayed on the ui
